nncp-cfgnew

$ nncp-cfgnew [options] [-nocomments] > new.hjson

Generate new node configuration: private keys, example configuration file and print it to stdout. You must use this command when you setup the new node. -nocomments will create configuration file without descriptive huge comments – useful for advanced users.

Pay attention that private keys generation consumes an entropy from your operating system.
