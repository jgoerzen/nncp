nncp-freq

$ nncp-freq [options] NODE:SRC [DST]

Send file request to NODE, asking it to send its SRC file from freq.path directory to our node under DST filename in our incoming one. If DST is not specified, then last element of SRC will be used.

If notification is enabled on the remote side for file request, then it will sent simple letter after successful file queuing.
