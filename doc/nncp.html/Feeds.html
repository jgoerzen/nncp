Integration with Web feeds

RSS and Atom feeds could be collected using rss2email program. It converts all incoming feed entries to email messages. Read about how to integration Postfix with email. rss2email could be run in a cron, to collect feeds without any user interaction. Also this program supports ETags and won’t pollute the channel if remote server supports them too.

After installing rss2email, create configuration file: 

$ r2e new rss-robot@address.com

and add feeds you want to retrieve: 

$ r2e add https://git.cypherpunks.ru/cgit.cgi/nncp.git/atom/?h=master

and run the process: 

$ r2e run
