Релиз 5.1.0

  • nncp-file может отправлять директории, автоматически на лету создавая pax архив.
  • Во время создания исходящих сообщений проверяется наличие свободного места на файловой системе.
  • freq, freqminsize, freqchunked опции конфигурационного файла заменены на структуру freq: {path: ..., minsize: ..., chunked: ...}.
  • Добавлена freq.maxsize опция конфигурационного файл, запрещающая ответ на файловый запрос больше заданного размера.
  • Возможность оповещения об успешно выполненных командах (exec) через notify.exec опцию конфигурационного файла.
