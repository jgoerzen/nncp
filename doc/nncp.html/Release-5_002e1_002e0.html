Release 5.1.0

  • nncp-file can send directories, automatically creating pax archive on the fly.
  • Free disk space is checked during outbound packets creation.
  • freq, freqminsize, freqchunked configuration file options replaced with the structure: freq: {path: ..., minsize: ..., chunked: ...}.
  • Added freq.maxsize configuration file option, forbidding of freq sending larger than specified size.
  • Ability to notify about successfully executed commands (exec) with notify.exec configuration file option.
