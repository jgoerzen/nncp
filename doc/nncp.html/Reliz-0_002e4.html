Релиз 0.4

  • Небольшое исправление в nncp-call, nncp-caller, nncp-daemon: иногда они могли падать с segmentation fault ошибкой (данные не терялись).
  • nncp-newnode переименована в nncp-newcfg – это короче и удобнее для использования.
  • Появилась команда nncp-mincfg: вспомогательная утилита позволяющая создать минималистичный урезанный конфигурационный файл без приватных ключей, что полезно во время использования nncp-xfer.
