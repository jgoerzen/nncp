nncp-stat

% nncp-stat [options] [-node NODE]

Print current spool statistics about unsent and unprocessed packets. For each node (unless -node specified) and each niceness level there will be printed how many packets (with the total size) are in inbound (Rx) and outbound (Tx) queues.
